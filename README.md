# AI Support Platform API

A production-style backend API for an AI-powered customer support platform built with NestJS, PostgreSQL, Redis, BullMQ, and OpenAI.

This project is created as a professional backend portfolio project to demonstrate scalable API architecture, authentication, background jobs, AI integrations, and clean backend engineering practices.

## Features

- JWT authentication
- Role-based access control
- Organization/workspace structure
- User management
- AI chat API
- Conversation history
- Redis caching
- Background jobs with BullMQ
- Webhook handling
- PostgreSQL database
- Swagger API documentation
- Docker-based local development

## Tech Stack

- Node.js
- TypeScript
- NestJS
- PostgreSQL
- Redis
- BullMQ
- Docker
- Swagger/OpenAPI
- OpenAI API
